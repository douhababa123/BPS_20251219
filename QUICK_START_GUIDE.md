# 🚀 能力画像模块快速上手指南

## ⏱️ 5分钟快速体验

### 步骤1：启动应用（30秒）
```bash
npm run dev
```
访问：`http://localhost:5173`

---

### 步骤2：导入数据（2分钟）

#### 2.1 导入能力定义
1. 点击左侧 **"数据导入(新)"**
2. 选择 **"能力定义导入 Skill Definition Import"**（绿色卡片）
3. 上传Excel文件（包含39个技能，格式：编号 | 模块 | 类型 | 工程师）
4. 点击 **"开始解析"**
5. 点击 **"开始导入"**
6. 等待成功提示：`✅ 成功导入 39 个技能`

#### 2.2 导入能力评估
1. 选择 **"能力评估导入 Assessment Import"**（蓝色卡片）
2. 上传Excel文件（包含员工评估数据，横向多表头格式）
3. 点击 **"开始解析"**
4. 点击 **"开始导入"**
5. 等待成功提示：`✅ 成功导入 X 条评估数据`

> 💡 **提示**：如果没有Excel文件，Supabase已经包含测试数据。

---

### 步骤3：体验团队视图（1分钟）

1. 点击左侧 **"能力画像 Competency Profile"**
2. 查看顶部4个统计卡片：
   - 👥 员工总数
   - 🏆 技能总数
   - 📊 平均现状
   - 📈 总差距
3. 默认进入 **团队视图 → 差距分析**
4. 观察：
   - **左侧雷达图**：9大模块的现状（蓝色）vs 目标（橙色）
   - **右侧柱状图**：各模块的差距分布
5. 点击 **"技能级"** 按钮，查看Top 15技能的差距
6. 切换到 **"排名"** Tab，查看模块排名表（🥇🥈🥉）

---

### 步骤4：体验个人视图（1分钟）

1. 点击 **"个人视图 Personal View"**
2. 在下拉框中选择一个工程师（例如：`Wang Ning - SNa-PS`）
3. 观察：
   - **左侧雷达图**：该工程师在9大模块的能力分布
   - **右侧柱状图**：该工程师的差距分布
   - **底部提升建议**：Top 10待提升技能
4. 点击 **"技能"** 按钮，切换到39技能视图
5. 点击 **"技能级"** 按钮（右侧柱状图），查看技能级差距

---

### 步骤5：导出数据（30秒）

1. 点击顶部 **"导出"** 按钮（蓝色）
2. 自动下载CSV文件：`能力画像_团队_2025.csv` 或 `能力画像_个人_2025.csv`
3. 用Excel打开，查看详细数据

---

## 📊 数据解读技巧

### 团队视图 - 雷达图
```
蓝色区域 < 橙色区域 → 有差距，需要提升
蓝色区域 = 橙色区域 → 已达标
```

### 团队视图 - 排名表
```
排名第1 = Gap最大 = 最薄弱模块 → 优先改进
```

### 个人视图 - 提升建议
```
Gap值越大 → 越需要培训
现状L1 → 目标L4（Gap=3）→ 优先级最高
```

## ❓ 遇到问题？

### 问题1：没有数据显示
**解决**：
1. 确认已导入数据（步骤2）
2. 点击刷新按钮
3. 检查Supabase连接（访问"连接诊断"页面）

### 问题2：个人视图是空的
**解决**：在下拉框中选择工程师

### 问题3：图表显示不全
**解决**：
- 雷达图：鼠标悬停查看完整名称
- 柱状图：横向滚动查看所有数据
- 导出CSV查看详细数据
